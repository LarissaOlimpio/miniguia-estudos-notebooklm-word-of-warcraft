# 🛠️ Prompts Testados — NotebookLM

Este documento reúne alguns dos principais prompts utilizados durante os estudos sobre Balance Druid utilizando o NotebookLM.

---

# 🎯 Objetivo

Os prompts foram criados para:
- melhorar a qualidade das respostas;
- resumir conteúdos complexos;
- acelerar revisões;
- comparar builds;
- aprender mecânicas do jogo;
- identificar erros comuns;
- organizar conhecimento.

---

# ✅ Prompt 1 — Rotação Básica

## 📌 Objetivo
Aprender a rotação básica do Balance Druid.

### Prompt Utilizado

```txt
Explique de forma simples como funciona a rotação do Balance Druid para iniciantes.
```

### 📥 Resultado Obtido
A IA explicou:
- geração de Astral Power;
- manutenção de Moonfire e Sunfire;
- uso de Starsurge.

### 📈 Qualidade da Resposta
Boa para iniciantes, mas superficial para conteúdos avançados.

---

# ✅ Prompt 2 — Builds

## 📌 Objetivo
Comparar builds para diferentes conteúdos.

### Prompt Utilizado

```txt
seguindo essa árvore de talento que coloquei sobre o Wowhead não tem o convocar espíritos, dá uma olhada tbm na rotação indicada no Wowhead pra single target e pra mult target e refaz pra mim o guia de rotação
```

### 📥 Resultado Obtido
A IA mostrou:
- diferenças entre talentos;
- builds para dano em área;
- builds focadas em single target.

# 📚 Aprendizados Sobre Engenharia de Prompt

Durante os testes foi possível perceber que:

- prompts específicos geram respostas melhores;
- contexto adicional melhora muito os resultados;
- mencionar o tipo de conteúdo (Raid/Mythic+) ajuda a IA;
- prompts genéricos produzem respostas superficiais;

---

# ✅ Conclusão

Os testes de prompts ajudaram a melhorar:
- qualidade das respostas;
- velocidade de aprendizado;
- organização das informações;
- entendimento sobre o Balance Druid.
