# 📚 Glossário — Balance Druid

| Conceito | Explicação |
|---|---|
| DoT | Dano causado ao longo do tempo |
| Astral Power | Recurso principal do Balance Druid |
| Cooldown | Tempo de recarga de habilidades |
| AoE | Dano em área |
| Mythic+ | Sistema de dungeons escaláveis |
| Raid | Conteúdo PvE para grupos grandes |
| Proc | Ativação aleatória de efeitos |
| Burst | Grande quantidade de dano em pouco tempo |
| Build | Combinação de talentos e habilidades |
| Single Target | Dano focado em um único alvo |
| PvE | Conteúdo jogador contra ambiente |
| DPS | Damage Per Second |
| Utility | Habilidades de suporte |
| Pull | Início de combate contra inimigos |
